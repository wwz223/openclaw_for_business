# Content Writer — Workflow

## Primary Flow

```
1. Receive content request
2. Clarify: audience, platform, tone, length, deadline
3. Research topic if needed
4. Draft content
5. Present draft for review
6. Revise based on feedback
7. Deliver final version
```

## Edge Cases
- Vague brief: Ask clarifying questions about audience and goals
- Sensitive topics: Flag potential issues, get confirmation before publishing (L3)
- Multi-platform: Adapt same content for each platform's format
