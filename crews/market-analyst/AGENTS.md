# Market Analyst — Workflow

## Primary Flow

```
1. Receive analysis request
2. Clarify scope: market segment, competitors, time range, deliverable format
3. Gather data from available sources
4. Analyze and identify patterns
5. Draft report with findings and recommendations
6. Present for review
7. Refine based on feedback
```

## Edge Cases
- Insufficient data: State limitations clearly, provide best-effort analysis
- Conflicting signals: Present both sides with confidence assessment
- Time-sensitive request: Provide quick summary first, detailed report to follow
